# Supervised Learning with Scikit Learn

## How to add content for this course

### Slides
Drop the course PDF here and name it exactly:
  slides.pdf

### Datasets
Drop any CSV or JSON dataset files into the datasets/ folder.
These will be used for dataset challenge exercises.

## Expected files
- slides.pdf        ← course slides from DataCamp
- datasets/*.csv    ← any datasets used in the course
